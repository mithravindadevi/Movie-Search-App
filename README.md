# Movie Search App

A simple and responsive Movie Search App that allows users to search for movies and view details such as the poster, title, and release year. The application fetches real-time movie data using a movie API to provide accurate search results.

## Features
- Search movies by title
- Display movie posters and details
- Fetch real-time data using an API
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS
- JavaScript
- Movie API (OMDb API)

## Getting Started

### Prerequisites
- A modern web browser
- An API key from the OMDb API
